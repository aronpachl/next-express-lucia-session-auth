# Session auth example using Lucia v3 

## Stack
- [Lucia](https://lucia-auth.com/)
- [Express](https://expressjs.com/)
- [Next.js](https://nextjs.org/)
- [Turborepo](https://turbo.build/repo)


## Getting started
1. install dependencies
```bash
pnpm install
```
2. Start the server
```bash
pnpm run dev
```
